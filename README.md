## Overview

This tool automates archiving GitHub projects. Given a list of repository names, it:

1. Clones each repo from a source GitHub account (via HTTPS + token)
2. Creates the corresponding repo in `aymnms-archives` if it doesn't exist yet
3. Pushes the current branch via SSH
4. Deletes the local clone

No dependencies beyond `git` and `curl` — both available natively on macOS and Linux.

---

## Prerequisites

- `git`
- `curl`
- SSH key configured for GitHub (`~/.ssh/id_ed25519` or equivalent)
- A GitHub Personal Access Token with the **`repo`** scope

---

## Setup

Run the setup once to store your credentials locally:

```bash
./run.sh --setup
```

You will be prompted for:

| Field | Description |
|---|---|
| GitHub Personal Access Token | Used for HTTPS clone and API calls. Scope: `repo` |
| GitHub source username | Account to clone repos from (defaults to your local `git config user.name`) |

Credentials are saved in `.env` (git-ignored).

---

## Usage

```bash
# Archive one or more projects
./run.sh <project1> [project2] [...]

# Reconfigure credentials
./run.sh --setup

# Reconfigure then immediately archive
./run.sh --setup <project1> [project2] [...]
```

**Example:**

```bash
./run.sh morpion-server my-website old-api
```

---

## How it works

```
run.sh
  └── loads .env (GH_TOKEN, SOURCE_GITHUB_USER)
  └── calls push.sh <projects...>
        └── for each project:
              1. git clone https://oauth2:<token>@github.com/<source>/<project>
              2. curl GET  → check if aymnms-archives/<project> exists
              3. curl POST → create repo if it doesn't (public)
              4. git remote set to git@github.com:aymnms-archives/<project>.git
              5. git push -u origin <current-branch>
              6. rm -rf local clone
```

---

## Configuration

The `.env` file (auto-generated by `--setup`) contains:

```bash
GH_TOKEN="ghp_xxxxxxxxxxxxxxxxxxxx"   # Personal Access Token (scope: repo)
SOURCE_GITHUB_USER="aymnms"           # GitHub account to clone from
```

See `.env.example` for reference.
