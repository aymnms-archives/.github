## What is this?

**aymnms-archives** is a GitHub organization used to store old and inactive repositories from [@aymnms](https://github.com/aymnms).

These projects are no longer actively maintained. They are kept here as a historical record — proofs of concept, experiments, and projects built at various stages of learning.

## Archiving tool

Repositories are mirrored here using [github-archiver](https://github.com/aymnms/github-archiver), an open-source CLI tool that copies a GitHub repository in full — branches, tags, description, and topics included.
